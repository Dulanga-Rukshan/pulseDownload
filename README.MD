# Pulse

Pulse is a simple app I built to make it easier to stay in touch with people without having to start a full conversation.
Instead of messaging, you send quick signals like:

- 💭 Thinking of you  
- 🫂 Need a hug  
- 🔥 Proud of you  

They show up as notifications. No chat, no replies, no history.

---

## Why I Built It

I noticed that a lot of the time I wanted to reach out to someone, but opening a chat felt like too much.
Not because I had something big to say — just because messaging usually turns into a conversation, and that’s not always what I want.
So I built something simpler.

---

## What It Does

- Send quick emotional signals  
- No chat threads  
- No read receipts  
- No pressure to reply  
- Lightweight and minimal  

---

## How It Works

1. Open the app  
2. Select "+" a connection
3. Type the invite code for the connection you want to connect with (you have to get invite a person by sharing your code and when they install the app
they will have to send you their invite code, here you paste it and start sending the Pulses.)

That’s it.
